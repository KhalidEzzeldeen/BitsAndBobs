This is the top level of the examples distribution package 
for the book Programming Python 4th Edition.

--The "Examples" directory contains all the book example code.

--The "extensions" directory contains 3rd-party open source 
  extensions used by some of the book's examples (its PIL 
  package must be installed for some image-based GUI examples).

--The "changes" directory logs changes made to this package,
  and includes a snapshot of the book's suport website.

See the README file in Examples for further documentation.
